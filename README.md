# 🏋️ FitTrackerAI

An intelligent web-based fitness tracker that provides personalized health insights based on user metrics. The application calculates BMI, calorie requirements, hydration goals, and generates customized diet and workout recommendations through an interactive dashboard.

---

## 🚀 Features

- 📊 BMI & Health Score Calculator
- 🔥 Daily Calorie Requirement Estimation
- 💧 Personalized Water Intake Recommendation
- 🥗 Diet Plans based on Fitness Goals
- 💪 Workout Recommendations
- ⚠️ Health Risk Alerts
- 📈 Interactive Progress Charts
- 📱 Fully Responsive User Interface

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Charts:** HTML Canvas
- **Architecture:** Client-side Web Application

## ⚙️ Health Metrics

The application calculates:

- Body Mass Index (BMI)
- BMI Category
- Basal Metabolic Rate (BMR)
- Daily Maintenance Calories
- Goal-based Target Calories
- Daily Water Intake
- Overall Health Score

---

## 🎯 Personalized Recommendations

Based on user inputs such as:

- Age
- Gender
- Height
- Weight
- Activity Level
- Fitness Goal
- Diet Preference
- Lifestyle Factors

FitTrackerAI generates:

- Customized Diet Plans
- Workout Suggestions
- Hydration Goals
- Health Risk Alerts

## ▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/PR0801/FitTrackerAI.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

No installation or backend setup is required.
